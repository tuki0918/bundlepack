# Compatibility Fixtures

This directory contains reviewed `.bundlepack` files used to verify the shared
format across native implementations.

- `FormatV1.json` contains implementation-neutral constants checked by both native test suites.
- `Compatibility/macOS` contains fixtures generated by the native macOS implementation and opened by both platforms.
- Windows fixtures are generated during CI, uploaded as a temporary artifact, and opened by the macOS compatibility job.

The checked-in packages contain public test data only. Do not add production,
private, or manually created user packages to this directory.
